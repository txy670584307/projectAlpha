# projectAlpha 用户使用手册

## 产品简介

projectAlpha 是一个简洁高效的 Ticket 任务管理工具，支持通过标签对任务进行分类、筛选和管理。

## 快速开始

### 访问系统

打开浏览器，访问项目部署地址。

### 界面概览

系统采用三栏布局：

| 区域 | 功能 |
|------|------|
| 左侧栏 | 标签筛选区，显示所有标签及关联的 Ticket 数量 |
| 顶部栏 | 搜索框 + 状态切换（全部/未完成/已完成） |
| 主内容区 | Ticket 列表，展示所有任务卡片 |

## 功能说明

### 1. 创建 Ticket

1. 点击左上角的 **「+ 新建」** 按钮
2. 在弹出的表单中填写：
   - **标题**（必填）：描述任务内容，最多 200 字符
   - **描述**（可选）：补充任务详细信息
   - **标签**（可选）：可点击已有标签选择，也可手动输入新标签
3. 点击 **「提交」** 完成创建

### 2. 编辑 Ticket

1. 将鼠标悬停在 Ticket 卡片上
2. 点击右上角的 **编辑图标**
3. 修改内容后点击 **「提交」**

### 3. 完成/取消完成

- **完成任务**：点击 Ticket 左侧的圆形图标，图标变为蓝色勾选状态
- **取消完成**：再次点击蓝色勾选图标，恢复为未完成状态

已完成的 Ticket 标题会显示删除线，并降低透明度。

### 4. 删除 Ticket

1. 将鼠标悬停在 Ticket 卡片上
2. 点击右上角的 **删除图标**
3. 确认删除操作

> 删除后数据不可恢复，请谨慎操作。

### 5. 标签管理

#### 筛选标签

- 点击左侧标签栏中的任意标签，列表将仅显示包含该标签的 Ticket
- 可同时选中多个标签进行组合筛选
- 点击 **「清除筛选」** 恢复全部显示

#### 添加标签

在创建或编辑 Ticket 时：

- **选择已有标签**：点击标签选择区中已有的标签，已选中的标签会高亮显示
- **手动输入新标签**：在输入框中输入标签名称后按回车键，自动创建并添加

### 6. 搜索功能

- 在顶部搜索框中输入关键字，系统将实时搜索匹配标题的 Ticket
- 搜索支持模糊匹配，输入部分关键词即可找到相关结果
- 点击搜索框右侧的清除图标可清空搜索内容

### 7. 状态切换

使用顶部的状态切换器可以快速筛选不同状态的 Ticket：

| 选项 | 说明 |
|------|------|
| **全部** | 显示所有 Ticket（未完成 + 已完成） |
| **未完成** | 仅显示待处理的任务 |
| **已完成** | 仅显示已完成的务 |

## 快捷键（未来版本支持）

|--------|------|

## 常见问题

**Q: 标签名称是否区分大小写？**  
A: 不区分。"Tag" 和 "tag" 会被视为同一个标签。

**Q: 可以同时筛选多个标签吗？**  
A: 可以。点击多个标签即可进行组合筛选。

**Q: 删除 Ticket 后还能恢复吗？**  
A: 不能。删除操作不可逆，请在删除前确认。

**Q: 搜索支持搜索描述内容吗？**  
A: 当前仅支持搜索标题。

## 技术栈

- **后端**：FastAPI + SQLAlchemy + PostgreSQL
- **设计风格**：Apple-inspired 极简设计
