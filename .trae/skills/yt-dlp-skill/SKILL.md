# yt-dlp 视频下载器

## 功能介绍

该技能使用开源工具 yt-dlp 来下载各种视频网站的视频。yt-dlp 是一个强大的视频下载工具，支持超过 10,000+ 个视频网站。

## 支持的网站

- YouTube
- 哔哩哔哩 (B站)
- 爱奇艺
- 腾讯视频
- 优酷
- 抖音
- 快手
- 微博视频
- 以及其他 10,000+ 个视频网站

## 使用方法

### 基本用法

当用户提供视频链接时，直接调用下载脚本进行下载：

1. 接收用户提供的视频链接
2. 检查并安装 yt-dlp（如果未安装）
3. 执行下载命令
4. 显示下载进度和结果
5. 提供下载完成的文件路径

### 示例使用场景

**场景 1: 下载 YouTube 视频**
用户输入: "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

执行步骤:
- 调用 `scripts/download_video.py` 脚本
- 下载视频到当前目录
- 返回下载结果

**场景 2: 下载 B 站视频**
用户输入: "https://www.bilibili.com/video/BV1xx411c7mW"

执行步骤:
- 调用 `scripts/download_video.py` 脚本
- 下载视频到当前目录
- 返回下载结果

## 技术实现

### 核心脚本

**scripts/download_video.py**

功能:
- 自动检查并安装 yt-dlp
- 支持指定输出目录
- 支持指定视频质量
- 显示详细的下载进度和结果

使用方式:
```bash
python scripts/download_video.py <video-url> [-o <output-directory>] [-q <quality>] [-v]
```

参数说明:
- `<video-url>`: 视频链接（必填）
- `-o <output-directory>`: 输出目录（可选）
- `-q <quality>`: 视频质量（可选，默认 best）
- `-v`: 显示详细信息（可选）

### 依赖项

- Python 3.6+
- yt-dlp（会自动安装）

## 注意事项

1. 下载视频时请遵守相关网站的使用条款和版权规定
2. 某些网站可能需要登录才能下载视频，这种情况下可能需要额外的认证信息
3. 大型视频文件可能需要较长的下载时间，请耐心等待
4. 下载速度取决于网络连接质量和视频服务器的响应速度

## 故障排除

### 常见问题

1. **无法安装 yt-dlp**
   - 解决方案: 手动安装 yt-dlp: `pip install yt-dlp`

2. **下载失败**
   - 检查网络连接
   - 确认视频链接是否有效
   - 尝试使用 `-v` 参数查看详细错误信息

3. **视频质量问题**
   - 使用 `-q` 参数指定质量，例如: `-q bestvideo+bestaudio`

4. **文件路径问题**
   - 使用 `-o` 参数指定具体的输出目录

## 高级功能

### 支持的视频格式

yt-dlp 支持多种视频格式，包括但不限于:
- MP4
- WebM
- MKV
- AVI

### 质量选择

常见的质量参数:
- `best`: 最佳质量（默认）
- `bestvideo+bestaudio`: 最佳视频+最佳音频
- `worst`: 最低质量
- `144p`, `240p`, `360p`, `480p`, `720p`, `1080p`: 指定分辨率

### 批量下载

对于播放列表或多个视频链接，可以多次调用下载脚本。
